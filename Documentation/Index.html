Pointless Endpoints

API Documentation

Base URL

http://thepointlessendpoints.tryasp.net

This is just a dumb REST API. No auth. No nothing.

/compareBooleans

Compares two booleans. A totally necessary feature.

Parameters: first (bool), second (bool)

fetch(`http://thepointlessendpoints.tryasp.net/compareBooleans?first=true&second=false`).then(r => r.text()).then(d => console.log(d));
